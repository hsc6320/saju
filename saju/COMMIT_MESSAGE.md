# 변경 이력 (CHANGELOG)

## [최근 변경사항]

### 개인맞춤입력 정보 사주별 관리 기능 추가
- **사주별 개인맞춤입력 정보 저장/로드**: 
  - `SettingsStorageService`에 사주별 저장/로드 메서드 추가
  - `name`과 `birth`를 키로 사용하여 각 사주별로 개인맞춤입력 정보 관리
  - 이전 사용자의 개인맞춤입력 정보가 남아있던 문제 해결

- **서버 전송 시 현재 사주 정보와 연결된 정보만 전송**:
  - `GPTService.getFortuneInterpretation`에서 현재 사주 정보(`name`, `birth`)와 연결된 개인맞춤입력 정보만 로드
  - 이전 사용자의 개인맞춤입력 정보가 잘못 전송되던 문제 해결

- **사주 입력 화면 개선**:
  - 사주 저장 시 해당 사주의 개인맞춤입력 정보도 함께 저장
  - 기존 사주 편집 시 해당 사주의 개인맞춤입력 정보 자동 로드

- **사주 삭제 시 개인맞춤입력 정보도 함께 삭제**:
  - 사주 삭제 시 해당 사주의 개인맞춤입력 정보도 함께 삭제되도록 개선
  - 데이터 정합성 유지
