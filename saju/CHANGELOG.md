# 변경 이력 (CHANGELOG)

## [최근 변경사항]

### 사주 입력 화면 개선
- **UI 재구성**: 이미지 디자인에 맞게 사주 입력 화면 전체 레이아웃 재구성
  - 프로필 섹션 추가 (노란색 원형 아바타, "새 사주" 표시)
  - 섹션 제목 추가 ("기본 정보 섹션", "태어난날(양력)", "이름 이욱")
  - 타원형 버튼 스타일 적용 (보라색 배경)
  - DatePicker/TimePicker를 통한 날짜/시간 선택 기능
  - 이름 입력 필드에 아이콘과 문자 카운터(0/50) 추가
  - 하단 버튼 스타일 변경 ("사주 저장하고 선택", 별 아이콘)

- **양력/음력 선택 제거**: 양력으로 고정

- **출생 시각 기능 개선**:
  - 기본값을 "선택안함"으로 설정
  - "모름" 선택 옵션 추가
  - "선택안함" 상태로 저장 시도 시 알림창 표시
  - "모름" 또는 시간 선택 시에만 저장 가능

- **날짜 입력 개선**: 직접 입력 가능 (형식: "2025-01-01" 또는 "20250101")

- **Validation 개선**:
  - 이름 미입력 시 "이름을 입력해주세요." 메시지 표시
  - 날짜 미입력 시 "날짜를 입력해주세요." 메시지 표시
  - 출생 시각 "선택안함" 상태로 저장 시도 시 알림창 표시

### 개인맞춤입력 기능 이동
- **위치 변경**: 설정 화면에서 사주 입력 화면으로 이동
  - 이름 입력 필드 아래에 배치
  - A. 필수, B. 권장, C. 보조(선택), D. 민감(제한 입력), E. 기타사항(선택) 섹션 포함

- **직업 상태 옵션 추가**: "공무원" 추가

- **설정 화면 정리**: 개인맞춤입력 섹션 제거, 알림 설정만 유지

### 사주 목록 화면 개선
- **"사주 조회" 버튼 위치 변경**: 하단 버튼에서 "..." 메뉴로 이동
  - 메뉴 항목: "간지 조회", "수정", "삭제"

- **사주 삭제 기능 개선**:
  - 삭제 확인 다이얼로그 추가
  - 서버 대화내용 삭제 기능 추가
  - 삭제 성공/실패 메시지 표시
  - 로딩 인디케이터 표시

### 저장 및 선택 기능 개선
- **사주 저장 후 자동 선택**: 저장한 사주가 자동으로 선택된 상태로 HomeScreen에 표시
  - 사주 결과 계산 (간지, 대운, 십성)
  - 선택된 사주로 자동 저장

- **HomeScreen 이동**: 저장 후 "운세 대화" 화면으로 이동

### 설정 화면 개선
- **모달 시트로 변경**: `showModalBottomSheet`를 사용하여 아래에서 올라오는 모달로 표시
- **개인맞춤입력 제거**: 사주 입력 화면으로 이동

### 모델 수정
- **SajuInfo 모델**: `time` 필드를 nullable(`TimeOfDay?`)로 변경하여 "모름" 옵션 지원

### API 서비스 개선
- **GPTService**: 대화내용 삭제 API 추가 (`deleteChatHistory`)
  - 서버에서 대화내용 삭제 기능
  - 삭제 성공/실패 응답 처리
